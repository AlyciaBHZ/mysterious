## 📖 项目简介

**小六壬排盘** 是一个基于古老六壬术数的现代化在线工具。通过输入日期数字和时辰，即可快速生成六宫排盘结果，包含：

- 🏛️ **六宫布局**：大安、留连、速喜、赤口、小吉、空亡
- 🌿 **五行属性**：木、火、土、金、水、天空
- 🕐 **时辰对应**：子丑寅卯辰巳午未申酉戌亥
- 🐉 **神煞标注**：玄武、青龙、朱雀、白虎、腾蛇、勾陈

本工具采用精准的传统算法，配合现代化的界面设计，让古老的智慧焕发新生。

## 🔮 核心功能

### 排盘要素

- **六宫布局**: 大安、留连、速喜、赤口、小吉、空亡
- **五行对应**: 木、火、土、金、水、天空
- **时辰系统**: 十二时辰完整支持
- **神煞标注**: 玄武、青龙、朱雀、白虎、腾蛇、勾陈

### 计算逻辑

1. 输入 X1 (1-30) 计算天盘落点
2. 选择时辰 (X2) 计算自身落点
3. 自动生成六宫排盘结果
4. 标注落点位置和对应神煞

## 📖 使用方法

### 快速上手

访问 [https://mysterious.lexaverse.dev](https://mysterious.lexaverse.dev)，按照以下步骤操作：

#### 步骤 1：输入 X1 数值
在第一个输入框中输入 **1-30** 之间的数字。这个数字通常对应：
- 农历日期
- 事件序号
- 或其他需要占卜的数字

#### 步骤 2：选择时辰
从下拉菜单中选择对应的时辰：
- **子时** (23:00-1:00)
- **丑时** (1:00-3:00)
- **寅时** (3:00-5:00)
- ... 以此类推

#### 步骤 3：开始排盘
点击 **"开始排盘"** 按钮，系统会立即生成六宫排盘结果。

#### 步骤 4：查看结果
排盘结果会以 **3×2 宫格** 的形式展示：

```
┌─────────┬─────────┬─────────┐
│  留连   │  速喜   │  赤口   │
├─────────┼─────────┼─────────┤
│  大安   │  空亡   │  小吉   │
└─────────┴─────────┴─────────┘
```

每个宫格包含：
- 📛 **宫位名称**（如：大安、留连等）
- 🌿 **五行属性**（如：木、火、土等）
- 🕐 **对应时辰**（如：子时、丑时等）
- 🐉 **神煞标注**（如：青龙、白虎等）

**红色标注"自身"** 的宫位即为当前落点位置。

### 使用示例

**例 1：占卜今日运势**
- X1 输入：15（今天是农历十五）
- X2 选择：7（现在是午时）
- 点击排盘，查看自身落在哪个宫位

**例 2：事件吉凶**
- X1 输入：3（事件相关数字）
- X2 选择：当前时辰
- 根据自身所落宫位判断吉凶
